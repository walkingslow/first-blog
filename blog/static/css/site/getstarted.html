Get Started


GE continues to transform into a digital industrial company and the technology, tools, and processes we use continues to transform along with it. Having a shared vision for what how our digital touchpoints look and behave is pivotal for our success. As more people and products are added to

GE digital landscape, the challenge of ensuring coherent digital experiences across channels multiplies exponentially. Core Design addresses this challenge by providing responsive, accessible, and highly modular CSS components that can be used by anyone:
DEVELOPERS

1. Explore the documentation to familiarize yourself with the various Core Design components and patterns.

2. Check out the Github repository to get started with the Core Design CSS.

3. Explore community technical implementations

4. Visit the Core Design - Developers Forum on either Slack (channel: #coredesign) or Yammer and join the conversation!
DESIGNERS

1. Explore the documentation to familiarize yourself with the various Core Design components and patterns.

2. Download the Core Design UI Kit (v1.0.1) for Sketch and start exploring the symbol library.

3. Visit the Core Design - IxD/VisD Forum on either Slack (channel: #coredesign) or Yammer and join the conversation! Ask questions, receive critique, and share your work with other Core Design community members.
