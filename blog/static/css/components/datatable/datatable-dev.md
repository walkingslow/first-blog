

## ge-datatable

**Element HTML:**
```html
<!-- Primary HTML -->
<table>
  <thead>
    <tr><th></th></tr>
  </thead>
  
  <tbody>
    <tr><td></td></tr>
  </tbody>
  
  <tfoot>
    <tr><td></td></tr>
  </tfoot>
</table>
```
